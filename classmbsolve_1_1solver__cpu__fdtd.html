Detailed Description

template<unsigned int num_lvl, template< unsigned int > class density_algo>
class mbsolve::solver_cpu_fdtd< num_lvl, density_algo >

OpenMP solver for c-lvl systems using the FDTD method. The number of levels c can be chosen arbitrarily, but must be known at compile time.
